# SanNav GTA V Interactive Map

A GitHub Pages ready interactive map website for SanNav.

## Features
- GTAWeb-style sidebar layout
- Pan and zoom map controls
- Searchable marker database
- Category filters
- Toggleable roads, districts, and coordinate grid
- Route planner mode
- Fullscreen support
- Mobile responsive sidebar

## GitHub Pages Setup
1. Upload these files to your GitHub repository.
2. Go to Settings > Pages.
3. Set Source to `Deploy from a branch`.
4. Select your main branch and root folder.
5. Save and open your Pages URL.

## Notes
This is a custom SanNav-styled map interface. It does not use GTAWeb assets.
